== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install it directly from the WordPress plugin dashboard.  
2. Activate the plugin through the **Plugins** menu in WordPress.  
3. Add the shortcode `[elementor_loop_filter]` before or above your Elementor Loop Grid widget.  
4. Customize the search and sorting options as needed.
